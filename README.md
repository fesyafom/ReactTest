# Getting Started

###### Install npm dependencies
`npm install`

###### Run the node server
`node app.js`

###### Open link to view it in the browser
`http://localhost:3000`

# API

```
/api/authors        get the list of authors
/api/authors/{id}   get specific author
```

```
/api/books          get the list of books
/api/books/{id}     get specific books
```
